Drones Intelligent Parking Management System.

A parking management platform using drone cruising paired with deep learning

Project Background

With the increase in the number of vehicles, the search for parking spaces is causing trouble for users, therefore, combining the advantages of intelligent processing brought by drones and deep learning can bring convenience to users in parking.

Problem to Solve

With this system, users can easily find parking spaces and navigate to the destination for parking with a single click, which saves time in searching for an empty space and avoids congestion due to wayfinding.

Solution Approach

This project takes images of car parking spaces through drone cruises, uploads the images to a cloud platform, uses deep learning algorithms for image processing, and obtains an accurate distribution map of car parking spaces within the campus.

workflow:

  • Leveraged advanced technologies such as drone and Machine Learning.
  • Adopted Agile development methodologies to ensure efficient project management.
  • Conducted regular user research and feedback sessions to continuously optimize features.
